# Library-Management-System
Python Tkinter Library Management System using SQLite
Library Management System

Overview
This is a Python-based Library Management System with a graphical interface built using Tkinter and a database using SQLite.
It allows users to manage books efficiently by adding, updating, deleting, and issuing books.

Features
- Add new book records with Book Name, Book ID, Author, and Status.
- Update book details and track whether books are issued or available.
- Delete individual books or the entire inventory.
- Issue books to users using a Card ID system.
- View all book records in a table with easy selection.
- User-friendly interface suitable for small libraries or personal use.

Installation
1. Ensure Python 3 is installed.
2. Tkinter is required (usually comes pre-installed with Python).
3. Clone this repository or download the ZIP file:
git clone https://github.com/hariommishra-12/Library-Management-System.git
4. Navigate to the project folder.

How to Run
1. Open the project folder.
2. Run the Python script:
python main.py
3. The GUI window will open and the Library Management System is ready to use.

Project Structure
Library-Management-System/
- main.py           : Main Python script with Tkinter GUI
- README.md         : Project documentation
